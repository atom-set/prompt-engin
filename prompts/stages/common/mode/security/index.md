## 📝 扩展指南

### 如何添加新的安全权限规则

1. **创建规则文件**：
   - 文件命名：使用 kebab-case（如 `new-rule.md`）
   - 文件头：包含文件说明和规则来源注释
   - 文件位置：`rules/stages/common/mode/security/`

2. **更新索引**：
   - 在本文件中添加新模块的说明
