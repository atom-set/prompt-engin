# 项目管理领域 Skills

本目录包含与项目管理和规范相关的 AI 提示词规范。

## Skills 列表

### clean-principle.md - 项目清洁原则
- **描述**：项目清洁原则，避免将 AI 辅助开发工具和非业务相关的脚本混入项目核心代码
- **标签**：`project`, `clean`, `principles`
- **使用场景**：项目初始化、代码审查、工具脚本管理时

## 使用方式

```bash
# 查看所有项目管理相关的 skills
openskills list | grep "project/"

# 加载特定 skill
openskills read clean-principle
```
