## 📝 使用说明

### 推荐使用方式

1. **直接使用完整分享内容**：
   - 使用 `完整分享内容.md` 作为主要分享文档
   - 文档包含所有内容和图表，可直接用于演示

2. **图表转换**（可选）：
   - 如果演示平台不支持 Mermaid，需要转换为 PNG 图片
   - 参考 `图表清单和使用说明.md` 进行转换
   - 转换后的图片保存在 `images/` 目录

3. **创建演示文稿**：
   - 基于完整分享内容创建演示文稿
   - 使用 Google Slides、PowerPoint 或其他工具
   - 将图表转换为图片后插入演示文稿

### 如何在 Notebook 中使用

**详细指南**：请参考 [Notebook使用指南.md](./Notebook使用指南.md)

**快速开始**：

1. **Google Colab**：
   - 创建新的 Notebook
   - 添加 Markdown 单元格
   - 复制分享内容
   - 图表转换为图片后插入

2. **Jupyter Notebook**：
   - 创建新的 Notebook
   - 添加 Markdown 单元格
   - 复制分享内容
   - 图表使用相对路径引用

3. **推荐方案**：
   - 方案A：转换为图片后使用（兼容性最好）
   - 方案B：使用 HTML 渲染 Mermaid（需要安装依赖）

### 如何转换为演示文稿

1. **Google Slides**：
   - 基于完整分享内容创建演示文稿
   - 将图表转换为图片后插入
   - 使用幻灯片模板美化

2. **PowerPoint**：
   - 导入 Markdown 内容
   - 插入图表图片
   - 使用主题模板

3. **其他工具**：
   - Reveal.js：支持 Markdown 和 Mermaid
   - Marp：Markdown 演示工具
